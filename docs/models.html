About the Value Score

The Value Score measures performance per dollar, helping you find models that offer the best bang for your buck in faith-based applications.

Calculation: Value Score = (Overall Score ÷ Total Cost) × 100

A higher value score means better performance for the price. For example, a model with an 80% score costing $0.50 (Value: 160 pts/$) offers better value than a 90% model costing $1.20 (Value: 75 pts/$), even though the latter has higher absolute performance.

Note: Only models with evaluation usage data display a value score.
